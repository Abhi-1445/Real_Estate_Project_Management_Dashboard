# 🌍 Global Real Estate Sales & Project Management Dashboard

## 📌 Project Overview

This project is a **conceptual Power BI dashboard** that simulates how a **global real estate company** can monitor property sales, revenue, and project performance in a centralized system.

It demonstrates how data visualization can help stakeholders make **data-driven business decisions**.

⚠️ Note: This is a **sample project created for demonstration purposes**, not real-time data.

---

## 🎯 Objectives

* Build a global real estate analytics dashboard
* Track revenue, expenses, and income
* Analyze property sales and performance
* Provide insights across regions and clients

---

## 🛠️ Tools & Technologies

* Power BI
* Data Modeling
* DAX
* CSV Dataset

## 📈 Key Features

* 💰 Revenue (26M), Expense (9M), Income (17M) KPIs
* 🏗️ Total Properties Sold Tracking (44)
* 📅 Monthly Revenue Analysis
* 🌍 Country-wise Revenue Distribution
* 🏢 Property Type Performance (Condo, Apartment, etc.)
* 👥 Top Clients & Revenue Contribution
* 📊 Interactive Filters (Year-wise: 2022–2024)

---

## 📈 Insights Generated

* India and Mexico contribute highest revenue
* July shows peak sales performance
* Condo and Single-family properties generate more revenue
* Top clients significantly impact total revenue

---

## 💡 Business Value

* Centralized view of global real estate performance
* Helps in strategic decision-making
* Identifies high-performing regions and clients
* Improves sales planning and forecasting
